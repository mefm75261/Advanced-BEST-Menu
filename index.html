How to Read Your Dashboard
Standard Work & Performance Management

1
Welcome to the most recently version of the Standard Work System
The report includes both an overview and a breakdown by terminal [ITI, ATI, TMAZ, TPG, SPC, STI, SVTI, FIT, and BEST], showing the three KPIs. Maturity Index = Avg(Score obtained / Maximum Score), indicating the progress made toward standardization. Deployment Index = Cum(Deployed Areas / Total Areas), current status of the committed deployment. System Index = Cum(Maturity Index * Deployment Index), actual indicator of the terminal’s progress.
GIF 2 — guide-actual-2.gif
2
How to navigate in the report
To navigate through the pages of the report, expand it using the diagonal arrows in the lower-right corner. Then use the arrows in the lower-center to turn pages, or click the "1 of 10" link in the middle to jump directly to any page.
GIF 3 — guide-actual-3.gif
3
Tools in Global's View
The overview has a tooltip; depending on where you position the mouse, it will show you the values used to calculate the indicator. Under System Index it shows the three indicators for the three time frames; under Maturity it shows the average score; and under Deployment it shows the areas that have been implemented, the total counts, and a list of them—highlighted in red if an instance is missing and in white if all are implemented.
GIF 4 — guide-actual-4.gif
4
Tools in Terminal's View
In the terminal view, you have the same pop-up tools, with the added benefit that you can see more detailed information by instance. The terminal view also includes an area filter, which displays the KPIs for that specific area—such as whether all instances have been implemented or are still pending—on a daily, weekly, or monthly basis.
GIF 5 — guide-actual-5.gif
5
Pop-up tables triggered by buttons [Cyan: Show, Gray: Hide]
In the terminal view, there are two buttons next to title to show and hide the detail tables; these display all the evaluations and scores by question. Next to them is a ranking of the questions with the lowest average and a list of all the questions with their numbers; all of them are also filtered by subject area.
